# Criação de Layout
## Enunciado 3 | Criação de Layout

### Layout I
No tópico 1 do enunciado 3 recebemos um desafio para criar um layout de tamanho máximo 750px contendo as seções de topo, 2 menus laterais um a esquerda outra na direita e uma seção central de conteúdo que fosse responsivo a todos os tipos de telas. Conforme solicitado o layout fiz o mesmo utilizando HTML/CSS com grid layout para fácil organização e posicionamento das seções. As cores utilizadas e textos das seções foram os mesmos do layout na página do enunciado.

### Layout II
No tópico 2 recebemos um desafio para montar uma página com as seções de topo, menu principal, 2 menus laterais uma a esquerda outra na direita, seção principal no meio e um footer, conforme os tópicos foram avançando pelo 3,4 e 5 foi pedido incrementos na página como um menu lateral na esquerda e um formulário na seção principal. Conforme solicitado o layout fiz o mesmo utilizando HTML/CSS com grid layout e flexbox para fácil organização e posicionamento das seções. As cores utilizadas e textos das seções foram os mesmos do layout na página do enunciado.

### Desafio
No final da página do enunciado 3 foi pedido um desafio de média escolar na tabela, onde seria possível preencher um formulário com as informações do aluno e de suas notas e a página devolver os dados computados em forma de tabela indicando os alunos aprovados e reprovados. Conforme solicitado o layout fiz utilizando HTML/CSS flexbox e javascript para dar o dinamismo na página.
