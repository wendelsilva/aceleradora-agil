Desafio

Elabore um programa que solicita o nome, nota do primeiro semestre, nota do segundo semestre frequência (em %) de um número indeterminado de alunos. Após cada conjunto de valores informados deverá ser apresentada uma mensagem para o usuário, perguntando se ele deseja continuar informando dados. Após usuário terminar de informar os dados dos alunos, o programa deverá imprimir uma tabela,

Nome 1° Semestre 2° Semestre Nota Final Frequência
